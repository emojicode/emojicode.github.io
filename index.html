What exactly is Emojicode?

Emojicode is an open soruce, high-level, object orientated programming language consisting of emojis.

Emojicode allows you to build fast cross-platform* applications while having a lot of fun.

*To all reasonable platforms in world: UNIX.

Awesome concepts.

Emojicode has got all the awesome paradigms you love: Object-orientation, Optionals, Generics and Closures.

Portable.

Emojicode binarys run on the so called Real-Time Engine which works across platforms. You can use it on OS X, Ubuntu, Debian, Raspbian...

Small and fast.

The Emojicode Real-Time engine is not just a few times faster than common virtual machines, itâ€™s just about 100 kilobytes.

Packages.

Emojicode has got a read-to-use package system to access every part of the system.
