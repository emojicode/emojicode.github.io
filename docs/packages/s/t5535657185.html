Class 🍡 🐚Element⚪

🍡 is a class whose subclasses can enumerate other objects. The 🍡 itself is never initiated.

To get the items from an iterator call the 🔽 method on it until ❓ returns 👎, which indicates that there are no more items.

Enumerators conform 🔂 and return themselves in the 🍡 method.

Methods

🔽

 🐖 🔽 ➡️ ⚪

Returns the next object or ✨ if there are no elements left.

Caution

You should not rely on the return of Nothingness to indicate there are no more objects to enumerate. Nothigness could be an ordinary element of the enumerated object. Use ❓ instead to determine whether there are elements left.

The default implementation always returns ✨.

Subclasses of this class are encouraged to declare a more specific return type.
